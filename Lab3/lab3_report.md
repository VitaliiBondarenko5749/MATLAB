# Лабораторна робота №3

## Відновлення зображень

------------------------------------------------------------------------

##  Мета роботи

Дослідити методи відновлення перекручених зображень як за відсутності
шуму, так і за його наявності.

------------------------------------------------------------------------

##  Теоретичні відомості

Модель формування зображення:

s(x,y) = h(x,y) \* u(x,y) + n(x,y)

де: - u(x,y) --- вихідне зображення\
- h(x,y) --- функція розмиття (PSF)\
- n(x,y) --- шум

------------------------------------------------------------------------

##  Хід роботи

### 1. Завантаження зображення

``` matlab
I = imread('images/cameraman.tif');
imshow(I)
```

------------------------------------------------------------------------

### 2. Розмиття (motion blur)

``` matlab
PSF = fspecial('motion', 21, 0);
blurred = imfilter(I, PSF, 'conv', 'circular');
```

------------------------------------------------------------------------

### 3. Відновлення без шуму

``` matlab
restored = deconvwnr(blurred, PSF, 0);
```

------------------------------------------------------------------------

### 4. Додавання шуму

``` matlab
noisy = imnoise(blurred,'gaussian',0,0.001);
```

------------------------------------------------------------------------

### 5. Відновлення з шумом

``` matlab
restored_noise = deconvwnr(noisy, PSF, 0.01);
```

------------------------------------------------------------------------

##  Відповіді на питання

### Формування зображення

Зображення формується як згортка вихідного сигналу з функцією розмиття
плюс шум.

### Модель

s(x,y) = h(x,y) \* u(x,y) + n(x,y)

### Відновлення

Використовується деконволюція: deconvwnr()
